# Changelog

## 0.1.0

- Initial public-ready release.
- Convert PDF, DOCX, PPTX, XLSX, and XLS files to Markdown.
- Support single-file and batch uploads.
- Preview and edit converted Markdown.
- Build structured prompts from converted content.
- Include 9 built-in prompt templates plus custom templates.
- Estimate token counts with `tiktoken` when available, with a character-based fallback.
- Chunk large documents with configurable size and overlap.
- Export converted Markdown and generated prompts as Markdown, text, or ZIP files.
